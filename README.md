## Clone the project

```bash
git clone https://github.com/POOL4T7/shop.git
```
<br>

## Install dependencies for server/client
### Client

```bash
cd client
npm install
```
### Server

```bash
cd server
npm install
```

# Install dependencies

```bash
npm run install-packages
```
## Run the server && client simultaneously

```bash
# Front-End
cd client
npm start

# Back-End
cd server
npm run dev

```
